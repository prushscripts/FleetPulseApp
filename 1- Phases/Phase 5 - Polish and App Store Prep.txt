PART 1: PUSH NOTIFICATIONS
═══════════════════════════════════════════════════════

In app/(manager)/index.tsx, on first load:

import * as Notifications from 'expo-notifications'
import * as Device from 'expo-device'

const registerForPushNotifications = async () => {
  if (!Device.isDevice) return null
  
  const { status: existingStatus } = 
    await Notifications.getPermissionsAsync()
  
  let finalStatus = existingStatus
  if (existingStatus !== 'granted') {
    const { status } = 
      await Notifications.requestPermissionsAsync()
    finalStatus = status
  }
  
  if (finalStatus !== 'granted') return null
  
  const token = (await Notifications.getExpoPushTokenAsync()).data
  
  // Save token to Supabase profiles table
  await supabase
    .from('profiles')
    .update({ push_token: token })
    .eq('id', user.id)
  
  return token
}

Add push_token column to profiles table:
ALTER TABLE profiles 
ADD COLUMN IF NOT EXISTS push_token TEXT;

Handle notification taps:
Notifications.addNotificationResponseReceivedListener(
  response => {
    const data = response.notification.request.content.data
    if (data.type === 'inspection_failed') {
      router.push('/(manager)/inspections')
    }
  }
)

# FleetPulse — App Store Listing

## App Name
FleetPulse

## Subtitle (30 chars max)
Modern Fleet Management

## Description
FleetPulse is the modern fleet management platform 
built for logistics and delivery companies.

Track vehicles, manage drivers, conduct digital 
pre-trip inspections, and monitor your fleet health 
— all from one intelligent dashboard.

FEATURES:
- Real-time vehicle tracking and oil change alerts
- Digital pre-trip inspections with pass/fail checklists
- Driver management with territory assignment
- Fleet health score based on maintenance compliance
- Issue reporting directly from driver mobile devices
- Announcement system for management to drivers
- Instant notifications for failed inspections

Built for operations teams who need reliability, 
speed, and clarity in managing their fleet.

## Keywords
fleet management, truck inspection, vehicle tracking, 
driver management, logistics, delivery, maintenance,
